----------------------------------------------------------
PROJECT DESCRIPTION
----------------------------------------------------------

Complaint Management System is a web-based application
that allows users to register complaints online and track
their complaint status. The administrator can view,
manage, update, and resolve complaints through an admin
dashboard.

HOW TO RUN THE PROJECT
----------------------------------------------------------

1. Install XAMPP.

2. Copy the project folder (CTMS)
   into:

   C:\xampp\htdocs\

3. Start:

   • Apache
   • MySQL

4. Open phpMyAdmin

5. Create Database:

   complaint_management_system

6. Import:

   complaint_system.sql

7. Open Browser:

   http://localhost/ctms

8. Login using the admin or user account.
